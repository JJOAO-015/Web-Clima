# Web Clima

Aplicação web desenvolvida com React e Vite para consulta de clima atual e dados marítimos por cidade.

## Funcionalidades

- Busca por cidade  
- Temperatura  
- Sensação térmica  
- Índice UV  
- Dados marítimos  

## Tecnologias

- React  
- Vite  
- JavaScript  
- CSS
- 
## APIs

- Weather API  
- Open-Meteo Marine API  

## Requisitos

- Node.js

## Execução

```bash
git clone https://github.com/JJOAO-015/Web-Clima.git
cd Web-Clima
npm install
npm run dev
